# Diwali Sales Analysis

## Project Overview

This project analyzes Diwali sales data using Python to understand customer purchasing behavior and sales trends. The analysis focuses on data cleaning, exploratory data analysis (EDA), and visualization to identify meaningful insights that can support business decisions.

The project demonstrates practical data analysis skills using real-world sales data.

## Objectives

- Explore the Diwali sales dataset.
- Clean and preprocess the data.
- Handle missing values and duplicate records.
- Perform exploratory data analysis (EDA).
- Visualize customer and sales trends.
- Generate business insights from the data.


## Tools and Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook


## Project Workflow

1. Imported the dataset.
2. Explored the dataset structure.
3. Cleaned and prepared the data.
4. Performed exploratory data analysis.
5. Created visualizations to identify sales trends.
6. Summarized key findings.


## Key Insights

- Analyzed customer purchasing behavior.
- Identified top-performing product categories.
- Explored age and gender-wise purchasing patterns.
- Examined state-wise sales performance.
- Generated insights using data visualization.
